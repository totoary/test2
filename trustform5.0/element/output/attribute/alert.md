#   alert                       

## DESCRIPTION

alert 속성을 줄 경우 이벤트를 설정할 수 있다.
이벤트를 설정하면 해당 이벤트가 발생 했을 때 alert 에 설정한 내용이 alert 창 형태로 보여진다

{% code title="\[Static\]" %}
```markup
<alert> 

	<![CDATA[ 이곳에 내용을 입력하세요 ]]>  

</alert>
```
{% endcode %}
