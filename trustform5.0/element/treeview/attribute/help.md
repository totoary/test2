# help

## DESCRIPTION
디자이너에서 attribute 창에 help 속성을 설정해 주면 내용이 알림창 형태로 보여진다.

{% hint style="info" %}
help 속성은 다른 attribute 처럼 Element 안에 속성형태로 설정되는 것이 아니라 treeview 컨트롤 하위에 별도의 Element 형태로 추가 된다. 
{% endhint %}

## EXAMPLE

{% code title="\[Static\]" %}
```markup
<help>
    <![CDATA[ 이곳에 Help의 내용을 입력하세요 ]]> 
</help> 
```
{% endcode %}
