# focuscolor  

## DESCRIPTION

이 속성을 설정하지 않으면 기본값은 "#ce9e39" 이다.

* attribute 는 디자인 시점에 설정하는 속성이기 때문에 스크립트로 focuscolor 를 동적으로 제어하기 위해서는 focusColor property 를 사용한다.
