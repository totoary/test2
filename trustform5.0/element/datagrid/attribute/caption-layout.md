# caption-layout

## DESCRIPTION

* flow : datagrid 가 여러 장에 걸쳐 print 될 때 caption 영역을 모든 페이지 마다 print 한다.
* static : 모든 페이지 마다 print 하지 않는다.

  \(속성을 설정하지 않으면 기본값은 “static " 이다\)

## **VALUES**

* **flow / static\(default\)**
