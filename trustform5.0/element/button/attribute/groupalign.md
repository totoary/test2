#   groupalign                        

## DESCRIPTION

* top : 페이지의 위쪽에 적용할 수 있는 모양으로 자동으로 바뀜
* bottom : 페이지의 아래쪽에 적용할 수 있도록 모양이 자동으로 바뀜
* user : 사용자가 스타일을 직접 설정할 수 있도록 보통 버튼과 같이 사각형의 모양
