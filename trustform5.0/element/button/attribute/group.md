#   group                        

## DESCRIPTION
group 이 같은 버튼들 중 하나가 선택 되면 다른 버튼들은 선택이 해제된다. 
{% hint style="info" %}Tab 버튼의 용도로 많이 쓰인다.{% endhint %}

## EXAMPLE

{% code title="\[Static\]" %}
```markup
<button id="switchButton2" group="switchGroup" selected="true" style="left:40px; top:235px; width:70px; height:20px; ">  
```
{% endcode %}
