# ellipsis

## DESCRIPTION
secret 컨트롤의 disabled(비활성) 상태를 설정하는 속성이다.
{% hint style="info" %}
{% endhint %}

## VALUES
true / false

## EXAMPLE

{% code title="\[Static\]" %}
```markup
<secret id="secret1" ref="/root/Basic" disabled="true" imemode="disabled" 
style="left:105px; top:110px; width:100px; height:20px; "/> 
```
{% endcode %}
