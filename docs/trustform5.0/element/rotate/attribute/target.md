# target 

## DESCRIPTION

애니메이션 효과가 적용될 컨트롤의 id를 지정하는 속성이다.

## EXAMPLE

{% code title="\[Static\]" %}
```markup
<animation id="ani1">
    <scale id="scale1" during="1000" frame="30" target="input1" widthTo="200" heightTo="100"/>
</animation> 
```
{% endcode %}
