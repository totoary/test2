#   id                       

## DESCRIPTION

스크립트에서 컨트롤의 접근하기 위한 고유 이름을 지정한다. 
  
## EXAMPLE

{% code title="\[Static\]" %}
```markup
<animation id="ani1">
    <move id="move1" during="1000" frame="30" target="input11" xTo="300" yTo="300"/>
</animation>  
```
{% endcode %}
