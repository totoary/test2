# bind

## DESCRIPTION

bind 창에서 bind 를 설정한 다음 bind 를 사용할 output 컨트롤에 bind ID 를 설정하여 사용한다. bind 를 사용하면 calculate, constraint 등 편리한 기능을 사용할 수 있다.

