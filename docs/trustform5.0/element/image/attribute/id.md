# id

## DESCRIPTION

서로 다른 종류의 컨트롤 일지라도 id 를 중복하여 사용할 수 없다.

## EXAMPLE

{% code title="\[Static\]" %}
```markup
<img id="TFimage" src="hint_image.jpg" style="left:100px; top:255px; 
width:285px; height:100px; background-stretch:stretch; "/> 
```
{% endcode %}
